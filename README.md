## Getting Started

`npm run server:inspect`

Run the MCP server from VS Code through `.vscode/mcp.json`

### Things you can do
* Open up a Copilot Chat and you're able to use it like `#create-user`
* You can also see your resources like 'List all users' you can get it from the MCP Server
* With Prompts, you can access them through the UI or through a `/` command in the Copilot chat i.e. `/mcp.test-msp-video-server.generate-fake..`
