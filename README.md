## Getting Started

`npm run server:inspect`

Run the MCP server from VS Code through `.vscode/mcp.json`

### Things you can do
* Tools - Open up a Copilot Chat and you're able to use the tool  `#create-user` to create a user.
* Resources - You can also see your resources like 'List all users' or 'retrieve user by id' so you can get it from the MCP Server and add it as context , etc.
* Prompts - With Prompts, you can access them through the UI or through a `/` command in the Copilot chat i.e. `/mcp.test-msp-video-server.generate-fake..` to leverage a predetermined prompt with bits of data you need.
